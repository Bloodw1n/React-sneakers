# React-sneakers
### Online store

Stack:
- ReactJS + хуки;
- React Router;
- Axios;
- SCSS;
- React Context;
- React Skeleton;
- MockAPI.

Steps to start the project:
#### `git clone https://github.com/Bloodw1n/React-sneakers.git`
#### `cd React-sneakers`
#### `npm install`
#### `npm run build`
#### `npm run start`
