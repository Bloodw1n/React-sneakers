# React-sneakers
### Интернет-магазин кроссовок

Стек:
- ReactJS + хуки
- React Router
- Axios
- SCSS
- React Context
- React Skeleton
- MockAPI

Подробный порядок действий для запуска проекта:
#### `git clone https://github.com/Bloodw1n/React-sneakers.git`
#### `cd client`
#### `npm init`
#### `npm install`
#### `npm run build`
#### `nppm run start`
